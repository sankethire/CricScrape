# CRIC$CRAP
### A scraper which provides ODI cricket player runs yearwise 

### How to run
* If "ODI_player_scores.csv" is occupied by the data run the following commands:
```
rm ODI_player_scores.csv
python3 cricket_players.py
```
* Otherwise run:
```
python3 cricket_players.py
```
### Tools used:
* BeautifulSoup
* Requests
* LXML
* CSV
